Project Name:
قال رسول الله

Project Type:
Mobile application built using Flutter.

Project Description:
"قال رسول الله" is a modern Islamic mobile application designed to help users strengthen their relationship with Islam through daily worship tools, Quranic and prophetic content, reminders, and spiritual engagement features. The app focuses on simplicity, beautiful user experience, performance, and encouraging consistent Islamic habits.

Target Users:
- Muslims of all ages.
- Students and adults.
- Users who want a complete daily Islamic companion.

Main Goals:
1. Help users maintain Islamic habits daily.
2. Make Islamic content easily accessible.
3. Increase engagement using streaks and progress tracking.
4. Provide a clean and modern user experience.

Tech Stack:
- Flutter
- Dart
- Clean Architecture
- Cubit / Bloc for state management / Riverpod
- Local storage:
   - Hive
   - SharedPreferences
- JSON assets for static data
- Geolocator and Compass packages for Qibla
- Optional Firebase integration later
